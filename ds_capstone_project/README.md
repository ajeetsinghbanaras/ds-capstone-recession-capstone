
# Economic Indicators & Recession Prediction – Data Science Capstone

This repository contains a complete, self-contained example capstone project.  
You can upload this structure directly to a GitHub repository and link it in your assignment.

## Contents

- `data/economic_indicators.csv` – synthetic macro-economic dataset.
- `notebooks/01_data_collection_wrangling.ipynb` – loading + cleaning + feature engineering.
- `notebooks/02_eda_visuals.ipynb` – exploratory data analysis and visualisations.
- `notebooks/03_sql_eda.ipynb` – SQL-style queries using SQLite.
- `notebooks/04_folium_map.ipynb` – interactive map using Folium.
- `notebooks/05_plotly_dash.ipynb` – Plotly Dash dashboard example.
- `notebooks/06_predictive_model.ipynb` – RandomForest classifier for recession prediction.
- `presentation_outline.md` – text outline for your PPT/PDF slides.
- `requirements.txt` – list of Python packages used.

## Recommended Order

1. Run `01_data_collection_wrangling.ipynb`
2. Run `02_eda_visuals.ipynb`
3. Run `03_sql_eda.ipynb`
4. Run `04_folium_map.ipynb`
5. Run `05_plotly_dash.ipynb`
6. Run `06_predictive_model.ipynb`

You can take screenshots and copy text from these notebooks to build your final PDF / PPT presentation.
